    永灭之王 / 魔躯 — 角色档案

    Summary¶

    永灭之王并非独立角色，而是魔躯在小说第三百二十二章中公开报出的尊号之一。其完整自称为"诸界的圣律之主，混沌的驾驭者，永灭之王"。魔躯本质上是诞生于因果律奥秘之中的未知存在，与邪魔之主、邪性之魔均非同一实体。

    ⚠️ VERIFIED on 2026-06-14 via SearXNG (8 results) + 多站章节确认: "永灭之王"是魔躯的自称/名号，不是单独人物。起点中文、UU看书、黄金屋等四家站点均收录第三百二十二章《永灭之王》。

    🔑 核心身份架构¶

    永灭之王 = 魔躯的尊号¶

    维度 内容
    名号 "诸界的圣律之主，混沌的驾驭者，永灭之王"
    本质 诞生于因果律奥秘之中的未知之物
    章节出处 第三百二十二章《永灭之王》
    确认方式 魔躯自身报出名号 + 序列情报分析 + 顾青山确认

    非等价实体（重要区分）¶

    永灭之王 ≠ 顾青山
永灭之王 ≠ 邪魔之主
永灭之王 ≠ 邪性之魔
永灭之王 = 魔躯在此阶段公开报出的尊号

    📖 关键原文证据¶

    序列情报分析（顾青山攻击魔躯后，系统给出鉴定）：

    "你的剑命中了魔躯。凭借此名号的威能，你感应到了聚集在对方身上的无数奥秘。由于这些奥秘乃是对方身躯的构成部分，所以你无法掌握，但通过这件事，本序列已经探查出对方的身份。对方乃是诞生于因果律奥秘之中的未知之物。"

    魔躯报出名号：

    "你们有必要听闻我的名号，也有这个荣幸知道自己将死在何等尊贵的存在手中——我是诸界的圣律之主，混沌的驾驭者，永灭之王。"

    顾青山确认：

    "所以也是'永灭之王。'"

    🎯 目标与行为¶

    魔躯 / 永灭之王的核心目标是：

    1. 夺取六道轮回所蕴含的那个术："我来此只为取走六道轮回所蕴含的那个术——唯有它才配得上我的身份。"
    2. 毁灭六道相关存在："六道的圣人，乃至六道的一切都必须毁灭！"

    上述两者存在内在矛盾（先灭六道，后术如何诞生？），表明该实体与六道体系关系复杂，并非简单的协助者或合作者。

    ⚔️ 与其他势力的关系¶

    势力/实体 关系
    邪性之魔 / 无转之地魔物 目标不同、存在矛盾和冲突；至少三方独立运作
    邪魔之主 非同一实体（详见邪魔之主档案）
    六道体系 敌对，试图毁灭但同时又需要其孕育之术
