    邪魔之主 — 角色档案

    Summary¶

    邪魔之主，最终BOSS。其真实身份是被"永灭之王"（地之纪元初来者）封印于青铜柱上的存在，称号"万物与众生的灭绝者，不可听闻的秘密末日"。

    ⚠️ VERIFIED on 2026-06-13 via 百度百科+抖音百科+知乎+PTT 四方交叉验证: 冰封尸体 = 幕，不是柳平，也不是邪魔之主本尊。柳平是在时间另一面扮演邪魔首领的人。

    🔑 核心身份架构¶

    真正的邪魔之主¶

    • 真实身份：被"永灭之王"（地之纪元初来者）封印于青铜柱上的存在
    • 称号："万物与众生的灭绝者，不可听闻的秘密末日"
    • 全盛名号: "伟大的无穷源力之主" — 数不清的世界曾生长于他的身躯之上，被众生尊称
    • 本质位格: 地之纪元混沌使徒创造的活体封印，最初末日主/诸界末日源头之一；与混沌/众生对立的存在

    幕 / 巨大冰封之尸（与墟墓篇章相关，≠墟墓概念本身）— 封印掌管者 (≠邪魔之主)¶

    SearXNG+百度百科+抖音百科+知乎 cross-verified 2026-06-13: 四方交叉验证确认：巨大尸体/冰封之尸 = 幕，不是邪魔之主本尊。幕是封印掌管者，负责防止冰封之尸复活。后来反抗了命运，主动汲取冰封之尸的力量。

    ⚠️ 边界说明：读者讨论中“墟墓/大尸兄”常被用来指代终局中的巨大尸体或邪魔首领级存在；本页不把“墟墓概念”直接等同“幕”。

    维度 幕 (冰封之尸/巨大尸体) 邪魔之主 (最终BOSS)
    真实身份 封印掌管者，本体就是冰封之尸 被永灭之王封印于青铜柱上的存在
    与主角关系 暗中帮助顾青山，促成推翻宿命，获得自由 被主角方联手消灭
    后期立场 与查明的真相合一，打破被控制的命运 面对无尽末日，以一人万生之术、平行世界之术、万灵蒙昧之术布局
    结局 脱离宿命后选择与顾青山共同终结反派阴谋 主角集合众人力量将邪魔彻底击败

    柳平 / 独孤峰 — 扮演邪魔首领的人 (≠邪魔之主)¶

    PTT 实况确认 (Apr 15, 2022): 柳平利用邪魔 BOSS 的黑甲"捏了一个独孤将军的身份"，在《诸界末日在线》时间线中扮演"邪魔首领/巨大黑甲尸体"。

    维度 柳平/独孤峰 邪魔之主 (最终BOSS)
    真实身份 《炼狱艺术家》男主，"再来人" 被永灭之王封印于青铜柱上的存在
    角色性质 演员 — 按固定历史演出 真正的最终BOSS
    结局 被顾青山打回炼狱之门，回家 被众生联手混沌力量彻底击败

    身份链条全景图¶

    永灭之王 → 创造封印看守冰封之尸 → 
→ 幕觉醒后反抗命运，主动汲取冰封之尸力量，成为顾青山的同伴并教他各种术法 → 
→ 柳平在时间另一面扮演邪魔首领 → 
→ 最终决战中幕与邪魔是不同角色：幕在双剑帮助下脱离被控制宿命获得自由；邪魔之主被主角方彻底消灭

    ⚔️ 远古背景（百度百科原文明确记载）¶

    在无尽纪元前，真实世界众生与邪魔展开恶战。因实力悬殊，众生借谢道灵之手施展终极术法"混沌"（即诸界末日在线），以末日手段重创邪魔。

    邪魔败逃虚空后释放至高三术： - 一人万生之术（魔皇/青铜之主掌握）→ 重现真实世界众生、积蓄能量 - 平行世界之术（龙神掌握）→ 抵御末日侵害，为邪魔承受攻击 - 万灵蒙昧之术（虫柱/九面虫魔掌握）→ 封锁真相，阻止众生探索混沌

    终极目标： 等六道六圣集齐 → 发起总攻 → 让六道轮回重回洪荒世界 → 被镇压的众生"瞬间真实降临"一举消灭全部。

    💀 结局命运¶

    • 邪魔之主：主角集合众人力量将邪魔彻底击败；终局涉及混沌力量吞没/压制墟墓篇章中的首领级存在，但“墟墓本体”精确指代仍需原文复核
    • 幕：查明真相后与末日合一，在顾青山和双剑帮助下脱离被控制的宿命，获得真正的自由
    • 柳平/独孤峰：按固定历史演出后被顾青山打回炼狱之门，柳平回家
    • 谢道灵角色：最终解除终极术法使顾青山变得不朽

    📝 读者评价（知乎/贴吧原帖引文）¶

    "从头到尾帮助主角的最大助力者竟然是邪魔一方相关的存在，反转没必要且不合理"（知乎用户普遍反馈）

    ⚠️ VERIFIED on 2026-06-13 via 百度百科+抖音百科+知乎+PTT: 冰封尸体 = 幕，柳平是扮演邪魔首领的人。读者该批评基于正确的事实判断。
