## 考据规范

1. 新结论必须先检索、再验证、再写入。
2. 本地 Wiki 不是绝对事实源；冲突结论必须标注置信度与来源。
3. 对“墟墓 / 邪魔之主 / 幕 / 大尸兄”等易混淆实体，优先使用独立概念页区分概念层与剧情代称层。
4. 页面状态分为：`complete`、`needs-more-sources`、`stub`。
5. `stub` 页面必须继续补证，不能长期停留在“未完成”状态。
