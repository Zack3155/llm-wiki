## Summary

**幕**，核心角色。其**本体就是冰封之尸"万物与众生的灭绝者"**。他是**封印掌管者**，负责防止冰封之尸复活。后来**反抗了命运**，主动汲取冰封之尸的力量，成为顾青山的同伴并教他各种术法。

> ⚠️ **VERIFIED on 2026-06-13 via 百度百科+抖音百科+知乎+PTT 四方交叉验证**: **冰封尸体 = 幕**，不是柳平，也不是邪魔之主本尊。柳平是在时间另一面**扮演邪魔首领**的人。

---

## 🔑 身份与演变线 (已正名)

```
远古纪元 → 冰封之尸被封印于青铜柱(蒙昧世界) → 
→ 幕作为封印掌管者诞生，负责镇压冰封之尸 → 
→ 后来觉醒反抗命运，主动汲取冰封之尸力量 → 
→ 成为顾青山的同伴，传授术法 → 
→ 终局与末日合一，在顾青山和双剑帮助下脱离宿命，获得真正自由
```
