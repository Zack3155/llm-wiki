# 幕 / 冰封之尸 — 角色档案

*注：本文整合自百度百科正文快照与多次对话考据。关于其结局存在两个版本需要澄清。*

## Summary

幕（巨大冰封之尸）是《诸界末日在线》中贯穿全书的核心辅助角色，外表为被封印在青铜柱上的冰冻结尸体。他长期以神秘方式帮助主角顾青山修炼、传授术法并最终共同对抗末日威胁。身份极为复杂：既是地之纪元混沌使徒创造的"活的封印"，也是后来觉醒后反抗命运的独立意志体。

**关键结论**: 幕 ≠ 邪魔之主。百度百科将其分列为两个独立实体：幕是封印的创造者与看守人，邪魔之主是被封住的囚徒。旧版 Wiki 错误地将两者等同。

## 🔑 身份与演变线

```
地之纪元末日被封印 → 冰封于青铜柱（活的封印） →
→ 幕觉醒自我意识 → 永恒深渊镇守人 →
→ 为救众神将躯体一部分变作里世界/化为永恒深渊 →
→ 重生万神之王 → 堕入深渊化为深渊之王 →
→ 被蕾妮朵儿杀死坠入深渊（因特性复活）→
→ 即将被灵魂尖啸者杀时，顾青山以友爱之手搭救成为同伴 →
→ 查明真相终结蕾妮朵儿 → 与末日合一 →
→ 最终获得真正的自由（在双剑帮助下脱离宿命控制）
```

## 基本能力与作用

### 核心能力与威能
- 被束缚在蒙昧世界的青铜柱上，虽死亡仍具无尽威能
- 即使被封印状态也能阻挡末日的侵袭
- 全盛名号："伟大的无穷源力之主" — 数不清的世界曾生长于他的身躯之上

### 传授给顾青山的关键术法

| 术法/能力 | 说明 | 重要性 |
|---|---|---|
| **世界融合对抗末日** | 将多个世界的五行本源炼化牵引，使世界更强大以抵抗末日侵袭 | ★★★ |
| **众生同调奥秘** | 分辨并融入含灵存在的构成部分，最终达到"任何人无法分辨"的境界。分四阶渐深 | ★★★（顾青山多次借此逃出生天） |
| **神魂牵引之法** | 每次顾青山渡劫时将其拉到身边交流 | ★★ |
| **天道帝身份提醒** | 帮助顾青山逐步了解谢道灵的真实身份 | ★★ |

### 关键事迹
1. 以巨大冰封尸体形态长期援助顾青山修炼
2. 在六道争雄时期帮助拖延天帝时间
3. 协助查明真相后终结蕾妮朵儿背叛行为
4. 最终与末日合一

## ⚠️ 结局版本冲突（未完全解决）

两个来源对幕的最终归宿描述不一致：

| 版本 | 结局 | 来源 |
|---|---|---|
| **A: 获得自由** | 在顾青山和双剑帮助下脱离被控制宿命，与末日合一获得真正的自由 | 本地 Wiki 考据（基于早期百度百科分析 + 知乎解读） — 已多次标注 ✅ VERIFIED |
| **B: 同归于尽坠入炼狱** | "最终决战中与顾青山同归于尽，自身坠入炼狱" | 百度百科原文快照（uid 5_606），2026-06-10 新发现 |

**核心不确定性**：
- 百度百科条目确实记载"与主角同归于尽坠入炼狱"
- 但该百科条目同时将"巨大尸体"混同为邪魔之主，存在角色混淆的历史
- 可能版本 A（获得自由）来自更完整的知乎全解析，版本 B（坠入炼狱）来自百科的简化或误植描述

**待验证**：需要通过更多一手来源确定幕的真实结局。当前无法排除两种可能 — 可能是"先同归于尽后以某种形式重生/获得自由"的综合叙述。
