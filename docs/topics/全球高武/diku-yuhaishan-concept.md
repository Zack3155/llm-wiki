## 概述

本页整理《全球高武》中**地窟 / 地界**、**御海山**、**王战之地**与**界域之地**的关系。当前可确认的主结论是：**地窟 / 地界是与人界相对的巨大异界世界；御海山是地窟 / 地界的中心大陆；王战之地位于御海山核心，是与二王传说和人类武者遗迹有关的禁地。**

置信度暂定为 **0.79**，状态为 `needs-more-sources`。原因是：地窟 = 地界、御海山是中心、王战之地是禁地等主干设定有多源交叉支持；但“500亿平方公里”“36皇朝、72神宗”“108外域、108界域之地”等细粒度数据主要来自百科式摘要和读者梳理，尚未逐章原文核验。

## 二、地窟 / 地界的定位

### 1. 地窟的基本定义

地窟不是普通地下洞穴，而是《全球高武》中与人界相对的巨大异界。百科式摘要和起点简介都把“地窟入侵、武道崛起”作为故事核心背景：地球之外存在凶险地窟，妖族强者不断入侵人类世界，普通人必须走上武道之路才能自保。

因此，地窟在叙事中的作用包括：

- 人类世界的主要外部威胁来源；
- 妖族、王庭、地窟强者与地窟势力的根据地；
- 人类武者成长、征战、争夺资源的主要战场；
- 推动“武道崛起”、武科体系、军部体系和镇星城防线形成的外部压力源。

### 2. 地窟与地界的关系

当前证据支持“地窟 / 地界”作为同一世界结构的不同称呼。更稳妥的写法是：

> **地窟 = 地界 = 与人界相对的巨大异界世界。**

二者语义略有差别：

| 名称 | 常见语境 | 说明 |
|---|---|---|
| 地窟 | 剧情、战争、入侵、地窟武者 | 更偏“外部威胁/战场”语境 |
| 地界 | 世界观、地理结构、王庭与外域 | 更偏“世界设定/地图结构”语境 |
| 禁区 | 王战之地、御海山核心危险区域 | 有时特指核心危险区域，不宜直接等同整个地窟 |

## 三、御海山的定位

### 1. 御海山的基本定义

御海山是地窟 / 地界的中心大陆，也是地窟权力结构的核心。百科式条目与知乎整理均提到“地界中心为御海山”，起点资源站人物资料也多次出现“坐镇御海山守护人类”的表述，说明御海山是地窟核心防线与战略枢纽。

### 2. 御海山内部结构

当前可整理出如下结构，但细粒度数字需后续原文补证：

| 结构 | 当前整理 | 证据状态 |
|---|---|---|
| 四大王庭 | 天命王庭、天植王庭、守护王庭、万妖王庭 | 百科式条目与读者梳理一致，需原文补证 |
| 皇朝 | 御海山内存在多个皇朝；读者整理称有36皇朝 | 数字需原文补证 |
| 神宗 / 宗派 | 御海山内存在多个神宗或宗派；读者整理称有72神宗 | 数字需原文补证 |
| 真王殿 | 御海山核心统治机构之一 | 百科式条目提到，需原文补证 |
| 王战之地 | 御海山中心禁地 | 百科式条目 + 第560章原文支持 |

### 3. 御海山的剧情作用

御海山不仅是地理中心，也是战略中心。第560章原文中，张鹏提到“人类绝巅坐镇御海山，没办法言传身教，教导大家”，这说明御海山与人类防线、绝巅坐镇和地窟战局密切相关。

## 四、王战之地与界域之地

### 1. 王战之地的位置

王战之地位于御海山中心，是御海山内部最重要的禁地。百科式条目和知乎整理均称“御海山中心是禁地‘王战之地’”。

### 2. 王战之地的性质

第560章《王战之地》提供了较强的原文证据。张鹏介绍王战之地时提到：

- 王战之地是方平一行人的目的地；
- 它可能由二王开战、带领强者厮杀形成，也可能与人类武者有关；
- 原文使用“传闻”“也许”“无法准确推断”等措辞，说明其起源并非完全确定；
- 王战之地地形会变化，存在界壁与不同层级区域；
- 四大王庭、皇朝、宗派都会派人进入，是地窟天才与人类武者试炼、厮杀、争夺遗迹资源的场所。

因此，王战之地的稳妥结论是：

> **王战之地是御海山中心禁地，传统传闻认为它与二王之战有关，也可能涉及人类武者遗迹；但其起源细节尚未完全确定。**

### 3. 界域之地

界域之地通常被读者梳理为外域之间、围绕御海山分布的地理单元。知乎整理称“外围是一百零八外域，外域和外域之间是界域之地”。该内容与地窟环形结构相符，但目前主要依赖读者梳理，尚未在本页完成逐章原文核验。

## 六、常见误区

| 误区 | 更稳妥说法 |
|---|---|
| 地窟只是地下洞穴 | 地窟是与人界相对的巨大异界世界，不是单一洞穴 |
| 御海山只是一座山 | 御海山是地窟 / 地界中心大陆，承载王庭、皇朝、神宗与王战之地 |
| 王战之地就是御海山 | 王战之地属于御海山，是御海山中心的禁地 |
| 王战之地起源已完全确定 | 原文仍用“传闻”“也许”“无法准确推断”，只能确认其与二王传说、人类遗迹和试炼场有关 |
| 禁区一定等于整个地窟 | “禁区”有时指王战之地或御海山核心危险区域，不宜直接等同整个地窟 |
