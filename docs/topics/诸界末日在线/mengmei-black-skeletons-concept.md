## 概述

**蒙昧世界中的黑色骷髅目前只能被谨慎判定为“蒙昧世界中的群体性邪魔/邪恶灵体残骸”，不能提升为某个具名隐藏角色。** 现有章节显示它们成群出现、被巨大尸体击杀后会从泥土中重新冒出；但与“巨大尸体/幕/邪魔之主”之间的读者混淆仍未完全消除，因此本页保持 `needs-more-sources`。

## 证据链

### 1. 章节描写

第 39 章《时空乱流》写到：

- 青铜巨柱下，无数黑色骷髅来回走动，想往青铜巨柱上爬。
- 巨大尸体往下喷出一口气，墨绿之火把下方所有黑色骷髅烧干净。
- 随后“青铜巨柱下面的泥土中，一个个黑色骷颅冒出头来”，很快又遍布空间。

第 603 章《世界之墓！》也写到：

- “整个大地，处处都是徘徊的黑色骷髅。”
- 顾青山和巨大尸体对话时，巨大尸体提到“怪物已经死了”。

### 2. 与“巨大尸体/幕”区分

巨大尸体是蒙昧世界中真正有身份的核心存在。中文资料与读者梳理中常将巨大尸体与“幕 / 邪魔之主 / 冰封尸体”联系起来。

因此，现有证据更支持：

```text
黑色骷髅 = 蒙昧世界中的低阶/群体邪魔残骸
巨大尸体 = 另有身份的核心存在，可能与幕/邪魔之主相关
```

## 结论

**当前判定：邪魔/邪恶灵体残骸，群体性存在，无单独姓名。**

需要注意：目前能直接证明的是“黑色骷髅是蒙昧世界中的邪魔类群体/残骸”，但章节没有用一句明确台词说“它们就是某某”。所以这里不把它们提升为某个具名角色。
