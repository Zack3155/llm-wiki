## 概述

**万界俯视者**是《诸界末日在线》中登场的神秘至高位格存在。正文给予的完整名号为**"真古之魔·万界俯视者"**——其中"真古之魔"为本体名号，"万界俯视者"为称号/尊称。

⚠️ 第1677章明确说明：英灵殿主对顾青山表示"我们只知道它的名号，但它究竟是什么……又有谁能知道"，因此**更深层本质在正文中未被完全揭开**。本页记录的是可确认到名号层级的信息，非最终本质。

## 一、基本身份确认

| 项目 | 内容 |
|------|------|
| 本体名号 | 真古之魔 |
| 称号/别称 | 万界俯视者 |
| 出处 | 《诸界末日在线》（烟火成城著） |
| 类型 | 远古高位格存在（非六道体系内普通圣人） |
| 与顾青山关系 | 获得其青睐，传授"灵魂俯视"之力；后期主动破圣界铁律助之 |

### 核心等式

```
万界俯视者 = 真古之魔（名号等价）
```

并非：
- 万界俯视者 ≠ 顾青山（顾青山仅获得其青睐/力量）
- 万界俯视者 ≠ 邪魔之主（邪魔之主属"永灭之王/万物与众生的灭绝者"线）
- 万界俯视者 ≠ 圣界本身（顾青山同时问"万界俯视者是谁？圣界又是什么？"，说明非同一存在）

## 二、证据链

### 第1676章《万界俯视者》——首次正式出场

顾青山在血海巨柱前喊出完整召唤名号：

> "响应我的呼唤，重新降临于世——"
> "真古之魔，万界俯视者！"

召唤效果：
- 血水巨柱顶端降下**无数苍白之物（骸骨之颅）**
- **所有白骨落下将整个世界掩埋**，连长满竖瞳的黑暗怪物（世界归属之主级别）亦未能逃脱
- 英灵殿主事后评价："界灵们帮你建立了圣界之轮，而你从圣界上召唤了那位存在，它吃掉了你的敌人"

能力层级判定：
```
至少可从圣界层面介入相位世界
召唤覆盖世界的白骨/骸骨之颅
直接抹除世界归属之主级别的敌人
```

### 第1677章《灵魂俯视》——名号确认 + 本质未明

顾青山追问身份，英灵殿主明确回答"只知道它的名号，但它究竟是什么……又有谁能知道？"。

战神界面系统提示：
> "你获得了**真古之魔·万界俯视者**的青睐。"
> "对方把一种灵的力量传授给了你。"

顾青山获得：
- **特殊奥秘：灵魂俯视**——当众生的情绪产生波动，可以窥见对方灵魂中隐藏的秘密
- 顾青山自述："原来真古之魔竟然拥有这样的能力，任何人的心念都瞒不过它"

### 第2191章《血海之门》——后期关键补充

万界俯视者在血海中再度发声：
> "去你想去的地方。"

关键对话揭示深层信息：
> "顾青山，我已经**破了圣界的铁律**，但也仅此而已了，再也不可为你多做什么……"

由此推断：
- 万界俯视者与**圣界铁律**直接相关且曾受其约束
- 它有能力破坏圣界规则（但非无限出手）
- 与**圣界、血海、世界之门、根本虚空**等后期结构有深刻联系

## 三、能力总览

| 能力 | 描述 |
|------|------|
| **灵魂俯视** | 读取众生情绪波动中隐藏的心念/秘密 |
| **骸骨天降** | 从血水巨柱召唤无数苍白骸骨之颅，可掩埋整个世界 |
| **抹除高位存在** | 直接吃掉/消灭世界归属之主级别的敌人 |
| **破圣界铁律** | 有能力破坏/无视圣界规则约束 |
| **跨维度介入** | 从圣界层面介入相位世界 |
| **关注与授力** | 对顾青山进行关注并赋予特殊奥秘力量 |

## 四、与圣界的关系分析

第1676-1677章：万界俯视者**可通过圣界之轮从圣界召唤**，说明它存在于圣界或与之紧密关联的位置。英灵殿主称"想不到你能从圣界唤醒它！"。

第2191章后期：它能说"破了圣界的铁律"，暗示：
- 它是**受圣界规则约束的高层存在之一**，而非单纯的旁观者
- 可能属于：**圣界的高层级居民／远古封印中的存在／血海—圣界—世界体系交汇处的古老魔性存在**

但正文未明确说"它就是圣界意志"或"它是圣界本身"。

## 五、与邪魔之主线的关系区分

| 万界俯视者/真古之魔 | 邪魔之主 |
|---|---|
| 通过血海巨柱+圣界之轮召唤 | 被封印于青铜柱上 |
| 对顾青山持善意（授力/帮助） | 最终BOSS，敌对方 |
| 称"真古之魔·万界俯视者" | 称"永灭之王""万物与众生的灭绝者" |
| 破圣界铁律助主角 | 收割众生生命的魔王秩序 |

目前证据不支持两者同一。

## 六、已知分歧与不确定处

### ⚠️ 本质未明（核心不确定性）

正文明确告知读者：万界俯视者的**最深层本质**在剧情阶段未被揭示。"真古之魔·万界俯视者"确认到的是名号/称号层级，其最终起源、真实身份或是否与洪荒/根本虚空的其他至高存在重合，正文未给出定论。

### ⚠️ "圣界高层居民"假设 vs "世界体系外存在"假设

- 假设A：它是圣界内部的高层级存在，受铁律约束，因此能破坏铁律但非无所不能
- 假设B：它来自圣界之外（可能是血海/根本虚空源头），铁律是后来加诸其身的限制
- 正文未明确支持其中任何一侧
