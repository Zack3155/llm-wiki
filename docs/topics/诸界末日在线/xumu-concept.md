## 概述

墟墓在《诸界末日在线》中至少有两层含义：概念层上，它是混沌体系中的毁灭力量源头、混沌之灵的坟墓与终局冲突的核心对象；剧情代称层上，读者讨论常把“墟墓 / 巨大尸体 / 大尸兄 / 邪魔首领”关联到终局中的首领级存在。由于“概念层墟墓”与“剧情代称层墟墓”容易被混用，本页暂标为 `needs-more-sources`，并优先区分证据边界。

# 墟墓 · 概念档案

## 概念层：墟墓是什么

百度百科词条对“墟墓”给出较明确的设定说明：

- 墟墓是“一切混沌之灵的坟墓”；
- 墟墓是“混沌之中的力量源泉之一”；
- “混沌是杀死墟墓的力量”；
- “混沌”在词条中也被解释为真实世界众生耗尽力量后释放的终极术法。

因此，概念层可以谨慎概括为：

> 墟墓是混沌体系中的毁灭/虚无端核心对象，是混沌之灵归宿与力量源头之一；混沌则是能够杀死、折磨或压制墟墓的终极力量。

## 剧情代称层：墟墓、巨大尸体、大尸兄与邪魔首领

贴吧终局梳理帖中出现了“墟墓（也就是巨大尸体）”“老顾所遇到的墟墓（大尸兄），就是邪魔们的首领”等读者归纳。PTT 讨论则强调柳平在时间线中扮演邪魔首领/巨大黑甲尸体，按照固定历史演出，帮助谢道灵和顾青山完成历史使命。

因此，剧情代称层应写成“读者归纳/剧情代称”，而不是官方定义：

- “墟墓”有时被读者用来指代终局中的巨大尸体/大尸兄；
- “大尸兄/巨大尸体”可能与邪魔阵营首领级存在相关；
- “柳平/独孤峰”是扮演邪魔首领的人，不应直接等同于邪魔之主本体；
- “幕/冰封之尸”是封印掌管者，不应与邪魔之主本体混同。

## ⚠️ 当前考据分歧

⚠️ 误读/更正：本页不建议直接写“墟墓 = 邪魔之主 = 幕 = 大尸兄”。更稳妥的写法是：

1. 百度百科支持“墟墓”作为概念/力量源/混沌之灵坟墓；
2. 贴吧读者梳理支持“墟墓/巨大尸体/大尸兄/邪魔首领”之间的剧情代称关系；
3. PTT 讨论支持“柳平扮演邪魔首领/巨大黑甲尸体”的时间线解释；
4. 现有 Wiki 中“幕 / 冰封之尸”与“邪魔之主”是不同实体；
5. “混沌吞没墟墓本体”中的“墟墓本体”究竟是概念层墟墓、巨大尸体，还是邪魔首领级存在，仍需原文复核。
