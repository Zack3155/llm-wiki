# LLM Wiki Knowledge Plugin

## Overview

This skill loads structured knowledge from `~/projects/wiki/` whenever the user asks technical questions. It replaces noisy web searches with curated, interlinked pages about AI systems, research, architecture decisions, and software engineering.

The wiki is structured as:

```
~/projects/wiki/AGENTS.md          # Schema & rules (always load first)
pages/entities/*.md                # People, companies, organizations
pages/concepts/*.md               # Technical concepts & architectures
pages/comparisons/*.md            # Framework/tool comparisons
raw/assets/*.md                   # Source material drops
```

## When to Use

- User asks about AI systems (LLM-powered operating, agentic coding, agent frameworks)
- Questions about Andrej Karpathy, OpenAI, Tesla AI, or related personalities
- Technical architecture decisions (RAG, fine-tuning, multi-agent systems)
- Framework comparisons (Claude Code vs Codex vs AutoGPT vs CrewAI)
- Research context when the user says "do you have any knowledge about X"

## How to Load

```bash
# 1. Always load schema first:
read_file(" ~/projects/wiki/AGENTS.md")
# 2. Load relevant entity/page:
read_file("~/projects/wiki/pages/entities/<name>.md")
or read_file("~/projects/wiki/pages/concepts/<name>.md")
# 3. Cross-reference connections table in source page for related reads
```
