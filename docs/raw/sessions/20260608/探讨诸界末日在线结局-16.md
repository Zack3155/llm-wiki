## 🎯 标准动作四步流程

### 第1步：准备与站位
- 双脚与肩同宽，膝盖微屈，重心略前倾
- 站在后场中央稍偏后的位置，便于快速启动

### 第2步：侧身引拍蓄力
- 迅速转体**侧身**，非持拍手向后伸展平衡
- 持拍手向后上方举拍，身体呈 **"反弓"状态** 蓄力
- 右肘抬高至耳侧，大臂贴近脸颊，球拍引至颈后
- **屈膝蓄力**：背部微弓、腹部收紧，像压缩弹簧

### 第3步：发力击球（核心）
**发力链条：** 腿蹬地 → 腰部扭转 → 收腹 → 手臂下挥 → 手腕扣压

| 关键点 | 说明 |
|--------|------|
| **最佳击球点** | 身体前上方（右肩前约 0.5-1米） |
| **拍面角度** | 垂直或略前倾，确保杀球角度陡峭 |
| **手腕运用** | 击球瞬间迅速下压 **"扣压"** ，手指紧握拍柄 |
| **发力原则** | 重"快"轻"力"——动作越小速度越快 |

### 第4步：收拍回位
- 击球后顺势随挥至身体左下方卸力，避免受伤
- 持拍脚先着地屈膝缓冲，迅速回到中心区域准备下一拍
